================================================================================
                        BEHIND THE SCENES
================================================================================

What GitHub Actions does automatically:

[1] Setup Environment
    └─ Start Ubuntu virtual machine
    └─ Install Java JDK 17
    └─ Install Android SDK Build Tools
    └─ Cache dependencies (for faster builds)

[2] Checkout Code
    └─ Clone your repository
    └─ Checkout main branch

[3] Build Process
    └─ Make gradlew executable
    └─ Run: ./gradlew assembleDebug
    └─ Gradle downloads dependencies (~500 MB first time)
    └─ Kotlin compiler compiles all .kt files
    └─ Resource processor handles XML/images
    └─ R8/ProGuard optimizes code
    └─ DEX converter creates Dalvik bytecode
    └─ APK packager bundles everything
    └─ APK signer signs with debug certificate

[4] Upload Artifact
    └─ Compress APK into zip
    └─ Upload to GitHub storage
    └─ Make available in Actions tab

All of this happens WITHOUT touching your computer!

================================================================================
                        SECURITY & PRIVACY
================================================================================

Q: Is my code safe on GitHub?
A: Yes! You can choose:
   - Public repo: Anyone can see the code
   - Private repo: Only you can see it (still free!)

Q: What about the APK signing?
A: GitHub uses a debug certificate (for testing only)
   For Google Play Store, you need a release certificate

Q: Can others download my APK?
A: Only if they have access to your repository
   Private repos = private artifacts

Q: What data does GitHub see?
A: Only your source code (that you upload)
   No personal data, no phone access
