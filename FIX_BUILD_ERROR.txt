                    STEP 3: Update the Workflow File
================================================================================

I've updated the workflow file on your computer with better error handling.

Upload the new version:

1. Go to: https://github.com/maxin3820-jpg/Doodle

2. Click on: .github/workflows/build-apk.yml

3. Click the pencil icon (Edit) on the right

4. Delete everything and paste this:

---START COPYING FROM HERE---
name: Build Android APK

on:
  push:
    branches: [ main, master ]
  pull_request:
    branches: [ main, master ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
    - name: Checkout code
      uses: actions/checkout@v4
      
    - name: Set up JDK 17
      uses: actions/setup-java@v4
      with:
        java-version: '17'
        distribution: 'temurin'
        cache: gradle
    
    - name: Create local.properties
      run: |
        echo "sdk.dir=/usr/local/lib/android/sdk" > local.properties
        
    - name: Make gradlew executable
      run: chmod +x gradlew
      
    - name: Validate Gradle wrapper
      run: |
        ls -la gradlew
        file gradlew
      
    - name: Build Debug APK
      run: ./gradlew assembleDebug --stacktrace --info
      
    - name: Upload APK
      uses: actions/upload-artifact@v4
      with:
        name: doodle-debug-apk
        path: app/build/outputs/apk/debug/app-debug.apk
        
    - name: Upload build logs on failure
      if: failure()
      uses: actions/upload-artifact@v4
      with:
        name: build-logs
        path: |
          **/*.log
          **/build/reports/
---STOP COPYING HERE---

5. Scroll down and click "Commit changes"

- "Task 'assembleDebug' not found" = app/build.gradle.kts missing
