================================================================================
                        TROUBLESHOOTING
================================================================================

"Java is not recognized"
   → Restart command prompt after installing Java
   → Or set JAVA_HOME: setx JAVA_HOME "C:\Program Files\Eclipse Adoptium\..."

"SDK location not found"  
   → Edit local.properties, add: sdk.dir=C\:\\Android

"Build failed"
   → Run: clean.bat
   → Then: build.bat

More help in: BUILD_WITHOUT_ANDROID_STUDIO.md

================================================================================
                        WHAT IS DOODLE?
================================================================================

Doodle is a minimalist task manager with:
✓ Extremely fast (no splash screen, no loading)
✓ Offline-first (works without internet)
✓ Auto-save (never lose data)
✓ Customizable (themes, colors, fonts)
✓ No ads, no login, no cloud sync
✓ Material 3 Design (modern UI)
✓ Smooth animations (200-300ms)

Technology:
• Language: Kotlin
• UI: Jetpack Compose (Material 3)
• Architecture: MVVM
• Database: Room (SQLite)
• DI: Hilt
• Min SDK: Android 8.0 (API 26)
