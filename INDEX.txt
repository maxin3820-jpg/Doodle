================================================================================
                    WORKFLOW GUIDE
================================================================================

FIRST TIME SETUP:
   1. Read START_HERE.txt
   2. Run check-requirements.bat
   3. If fails, follow QUICK_START.md
   4. Run build.bat
   5. Run install.bat (or copy APK manually)

DEVELOPMENT WORKFLOW:
   1. Edit code in app/src/main/java/
   2. Run build.bat
   3. Run install.bat
   4. Test changes on device
   5. Repeat

TROUBLESHOOTING:
   1. Check START_HERE.txt section on troubleshooting
   2. See NO_ANDROID_STUDIO_SETUP.txt section 5
   3. Try clean.bat then build.bat again

================================================================================
                    QUICK COMMAND REFERENCE
================================================================================

BUILD COMMANDS:
   build.bat                        Build debug APK
   gradlew.bat assembleRelease      Build release APK
   gradlew.bat clean                Clean build
   gradlew.bat test                 Run tests

DEVICE COMMANDS:
   adb devices                      List connected devices
   adb install <apk>                Install APK
   adb uninstall com.doodle.app     Uninstall app
   adb logcat                       View device logs

PROJECT COMMANDS:
   gradlew.bat tasks                List all tasks
   gradlew.bat dependencies         Show dependencies
   gradlew.bat lint                 Check code quality

================================================================================
                    HELPFUL TIPS
================================================================================

✓ Always start with START_HERE.txt
✓ Run check-requirements.bat before building
✓ First build takes 5-10 minutes (downloads dependencies)
✓ Use clean.bat if build fails
✓ Keep local.properties configured
✓ Check phone has Android 8.0+ (API 26+)
✓ Enable USB debugging for adb install
✓ Subsequent builds are ~30 seconds
